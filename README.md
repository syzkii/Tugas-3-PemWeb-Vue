<b><h1>Tugas 3 Pemrograman Berbasis Web/h1></b>
<p>
Tugas Praktik 3 ini merupakan tugas terakhir dari mata kuliah Pemrograman Berbasis Web. Tujuan utama tugas ini adalah mengimplementasikan konsep Vue.js, termasuk penggunaan Vue Component, Template, data binding, directive, conditional rendering, array processing, filter, dan event handling untuk membangun aplikasi website pemesanan Bahan Ajar Universitas Terbuka. Mahasiswa diminta membuat aplikasi dengan struktur proyek Vue.js yang rapi, menampilkan daftar stok bahan ajar, fitur tracking delivery order, serta formulir input dengan validasi sederhana, dan menjelaskan pengerjaan melalui video. Tugas ini juga menekankan kreativitas dalam tampilan antarmuka dan pengalaman pengguna yang nyaman.
</p>
